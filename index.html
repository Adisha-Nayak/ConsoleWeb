How to set Visual Studio up?

Hi! Well, this is only the second time I'm using this... But I sure can explain how to set it up!

Step 1: Open a new folder. If you don't have an empty one, create a new folder.

Step 2: Create the index.html, style.css and main.js files in your folder by right clicking on the 'Explorer' area.

Step 3: Well, basically everything is done! Start coding! You can also add images into your folder if you want.

If you want to do things such as live testing and arranging your text to be neater, you can use the given extensions. There are hundreds of extensions you can download!

Click the last icon to find the extensions tab.

You can download new ones by clicking the 'Install' button.

What is a console? How to find it?

The console logs information associated with a web page: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript code running in the page context.

How to open console?

Right click any partof the web page that you have currently opened and click 'Inspect', lowermost button.

Now, check the right side of your screen and click 'Console'
